spaCy Text Redactor

Advanced Async Text Redaction Service

A text redaction service built with FastAPI and spaCy NLP. Features intelligent entity recognition, async background processing, comprehensive testing, and production-ready deployment. Designed to identify and redact sensitive information from text documents with detailed metadata tracking.

Python FastAPI spaCy asyncio Docker pytest NLP CI/CD

Project Overview

The spaCy Text Redactor is a backend service designed for enterprise-level text redaction needs. Built with modern Python technologies, it combines the power of spaCy's natural language processing with FastAPI's high-performance async capabilities to create a robust, scalable redaction solution.

This project demonstrates advanced software engineering practices including async programming, comprehensive testing strategies, containerized deployment, and CI/CD automation. The service is designed to handle sensitive data processing with intelligent entity recognition and detailed audit trails.

NLP Processing Pipeline

Entity Recognition & Redaction

The service uses spaCy's en_core_web_sm model for Named Entity Recognition (NER), combined with custom regex patterns for specialized data types. Each detected entity is replaced with [REDACTED] tokens while maintaining detailed metadata for audit purposes.

PERSON Names of people
ORG Organizations
GPE Geopolitical entities
DATE Date references
TIME Time references
EMAIL Email addresses
PHONE Phone numbers

Intelligent Inference System

Beyond simple pattern matching, the service includes a rule-based inference engine that provides contextual reasoning for each redaction decision. This adds an additional layer of intelligence to the redaction process, helping users understand why specific information was flagged.

FastAPI Route Handler

@app.post("/submit")
async def submit_text(
    text_submission: TextSubmission, background_tasks: BackgroundTasks
):
    try:
        job_id = str(uuid.uuid4())
        logger.info(f"Job {job_id} queued")
        job_store.add_job(
            job_id,
            "queued",
            RedactedResult(job_id=job_id, redacted_text="", metadata=[]),
        )
        background_tasks.add_task(run_redaction_task, job_id, text_submission.text)
        return JSONResponse(
            content={"job_id": job_id},
            status_code=202,
        )
    except Exception as e:
        logger.error(f"Error submitting job: {e}")
        return JSONResponse(
            content={"error": "Failed to submit job"},
            status_code=500,
        )

spaCy NLP Processing

def sanitize_text(text):
    doc = nlp(text)
    redacted_text = text
    metadata = []
    offset = 0  # Track offset due to replacement length changes

    for ent in doc.ents:
        if ent.label_ in ["PERSON", "ORG", "GPE", "DATE", "TIME"]:
            start, end = ent.start_char + offset, ent.end_char + offset
            original = redacted_text[start:end]
            metadata.append(
                {
                    "type": ent.label_,
                    "span": [start, end],
                    "text": original,
                }
            )
            # Replace entity with [REDACTED]
            redacted_text = redacted_text[:start] + "[REDACTED]" + redacted_text[end:]
            
            # Update offset for next replacements
            offset += len("[REDACTED]") - (end - start)

    metadata = annotate_redaction(metadata)
    return redacted_text, metadata

Async Background Task Processing

async def redact_text(job_id: str, text: str) -> Dict[str, Any]:
    # Simulate redaction process
    JobStatus(job_id=job_id, status="processing")
    await asyncio.sleep(2)  # Simulate processing time

    redacted_text, metadata = await asyncio.to_thread(sanitize_text, text)
    
    # Store result in job store
    result = {
        "status": "completed",
        "result": RedactedResult(
            job_id=job_id, redacted_text=redacted_text, metadata=metadata
        ),
    }
    job_store[job_id] = result
    JobStatus(job_id=job_id, status="completed")
    return result
