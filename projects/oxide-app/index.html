Oxide Webapp

Binary Analysis & Visualization Tool

A desktop and web application for binary analysis, reverse engineering, and malware research. Features interactive visualizations, statistical analysis, and disassembly capabilities built with modern web technologies and packaged as an Electron application.

Vue.js Nuxt.js Electron D3.js Chart.js Cytoscape.js Python PrimeVue Tailwind CSS

Project Overview

Oxide App represents the cutting edge of binary analysis tools, combining powerful reverse engineering capabilities with intuitive visual interfaces. Designed for security researchers, malware analysts, and reverse engineers, this application takes binary data and turns it into comprehensible visual representations.

Main Dashboard - Navigation interface showing all available tools

Core Visualization Modules

Binary Visualizer

Interactive grid-based visualization of raw binary data with color-coded byte values using D3.js viridis color scale. Features real-time hover tooltips showing decimal, hexadecimal, and binary representations.

Technologies: D3.js, Canvas API, Vue.js

Byte Histogram

Statistical frequency analysis of byte distribution with logarithmic scaling. Provides insights into data patterns and helps identify encrypted or compressed sections.

Technologies: Chart.js, Statistical Analysis

Byte N-grams Analysis

Matrix-based heatmap visualization of byte sequence patterns. Uses advanced matrix rendering to identify common byte patterns and sequences in binary data.

Technologies: chartjs-chart-matrix, Heatmap Algorithms

Call Graph Visualization

Interactive network visualization of function call relationships with hierarchical layouts. Shows incoming/outgoing call counts and directed graph representations.

Technologies: vis-network, Graph Theory

Control Flow Graph

Advanced program flow analysis using Cytoscape.js with function selector and interactive block visualization. Supports both control flow and call graph overlays.

Technologies: Cytoscape.js, Dagre Layout

Entropy Analysis

Statistical entropy analysis across memory addresses with line charts and detailed metrics tables. Identifies encrypted/compressed sections by entropy spikes.

Technologies: Chart.js, PrimeVue DataTable

Opcode Histogram

Assembly instruction frequency analysis with automatic logarithmic/linear scale switching. Helps identify compiler patterns and code characteristics.

Technologies: Chart.js, Assembly Analysis

Technical Architecture

Frontend Architecture

  • Nuxt.js 3 - Vue.js framework with SSR capabilities
  • Vue 3 Composition API - Reactive component architecture
  • PrimeVue - Enterprise-grade UI component library
  • Tailwind CSS - Utility-first CSS framework

Visualization Libraries

  • D3.js - Data-driven document manipulation for binary grids
  • Chart.js - Responsive charts for histograms and line graphs
  • Cytoscape.js - Graph theory library for control flow graphs
  • vis-network - Network visualization for call graphs
  • chartjs-chart-matrix - Matrix/heatmap chart extensions

Desktop Integration

  • Electron - Cross-platform desktop application framework
  • nuxt-electron - Nuxt.js integration with Electron
  • electron-builder - Application packaging and distribution

Backend & API

  • Python API Server - Backend server interacting with Oxide (Oxide-Formula)
  • RESTful API - HTTP-based communication protocol
  • Oxide Engine - Core analysis modules and disassembly

Key Features & Capabilities

Interactive Analysis

Real-time binary analysis with interactive tooltips, hover states, and dynamic data exploration. Users can navigate through binary structures with mouse interactions and keyboard shortcuts.

Export Capabilities

Export functionality supporting SVG and PNG formats for all visualizations. Charts and graphs can be exported with custom backgrounds and high-resolution settings for presentations and reports.

Advanced Graph Algorithms

Implementation of graph layout algorithms including Dagre for hierarchical layouts and force-directed positioning for optimal node arrangement in call graphs and control flow visualizations.

Statistical Analysis

Statistical analysis including entropy calculations, frequency distributions, n-gram analysis, and pattern recognition algorithms to identify anomalies and characteristics in binary data.

Development Highlights

Modern Development Practices

Built using modern JavaScript frameworks, component-based architecture for maintainability, and reactive programming patterns for optimal performance.

Cross-Platform Compatibility

Leverages Electron's cross-platform capabilities to deliver a consistent experience across Windows, macOS, and Linux while maintaining native desktop integration and performance.

Modular component design allows for easy extension and maintenance. Each visualization module is self-contained with its own data fetching, rendering, and export capabilities.

Installation & Usage

Development Setup

# Clone the repository git clone https://github.com/yourusername/oxide-app.git cd oxide-app # Install dependencies npm install cd client && npm install # Start development servers ./run.sh # Starts both client and API server

System Requirements

  • Node.js 16+ and npm/yarn package manager
  • Python 3.8+ for the analysis backend
  • Modern web browser with WebGL support
  • Minimum 4GB RAM for large binary analysis
  • Graphics card with hardware acceleration recommended
